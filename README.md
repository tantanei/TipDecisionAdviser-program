## 1. 제작 계기

이 프로그램은 Python 수업 3단원의 Lab 과제인 '팁 결정하기'에서 아이디어를 얻어 제작하였다. 해당 Lab은 사용자가 원하는 팁 비율을 선택하면 그 비율에 따라 팁 금액을 계산해 주는 단순 계산 프로그램이었다.

이 과제를 보면서, 이미 정해진 팁 비율로 금액만 계산하는 것보다 사용자의 상황과 만족도를 바탕으로 몇 퍼센트의 팁을 주는 것이 적절한지 추천해 주는 프로그램을 만들면 더 흥미로울 것이라고 생각했다. 그래서 서비스 품질, 직원 응대, 사용자의 만족도 등을 입력받아 적절한 팁 비율을 계산해 주는 "TipDecisionAdviser" 프로그램을 만들게 되었다.

수업 Lab은 프로그램의 주제를 정하는 계기가 되었지만, 프로그램 동작 방식, 실제 코드는 해당 Lab 코드를 참고하거나 가져온 것이 아니라 직접 설계하여 구현하였다. 따라서 이 README에서는 수업 Lab을 제작 계기에서만 언급하고, 코드 참고 출처로는 따로 기재하지 않았다.
