## 1. 제작 계기

이 프로그램은 Python 수업 3단원의 Lab 과제인 '팁 결정하기'에서 아이디어를 얻어 제작하였다. 해당 Lab은 사용자가 원하는 팁 비율을 선택하면 그 비율에 따라 팁 금액을 계산해 주는 단순 계산 프로그램이었다.

이 과제를 보면서, 이미 정해진 팁 비율로 금액만 계산하는 것보다 사용자의 상황과 만족도를 바탕으로 몇 퍼센트의 팁을 주는 것이 적절한지 추천해 주는 프로그램을 만들면 더 흥미로울 것이라고 생각했다. 그래서 서비스 품질, 직원 응대, 사용자의 만족도 등을 입력받아 적절한 팁 비율을 계산해 주는 "TipDecisionAdviser" 프로그램을 만들게 되었다.

수업 Lab은 프로그램의 주제를 정하는 계기가 되었지만, 프로그램 동작 방식, 실제 코드는 해당 Lab 코드를 참고하거나 가져온 것이 아니라 직접 설계하여 구현하였다. 따라서 이 README에서는 수업 Lab을 제작 계기에서만 언급하고, 코드 참고 출처로는 따로 기재하지 않았다.

## 2. 프로그램 설명 및 사용 방법

"TipDecisionAdviser"는 사용자가 입력한 서비스 만족도와 자신의 기분을 바탕으로 적절한 팁 비율을 추천해 주는 프로그램이다. 단순히 고정된 비율을 적용하는 것이 아니라, 서비스 유형과 사용자가 입력한 평가 점수를 반영하여 팁 비율을 계산한다. 또한 같은 점수를 입력하더라도 매번 완전히 동일한 결과만 나오지 않도록, 계산 과정에 작은 범위의 랜덤 조정값을 추가하였다.

이 프로그램에서는 서비스의 종류를 식당 홀서비스, 배달, 기타 서비스의 세 가지로 나누고, 각 유형마다 기본 팁 비율과 점수 반영 방식을 다르게 설정하였다. 식당 홀서비스를 하나의 예로 들면, 프로그램은 제공받은 서비스가 얼마나 만족스러웠는지, 오늘 자신의 기분은 어떤지, 서빙 직원은 만족스러웠는지를 차례대로 질문한다. 사용자는 각 질문에 대해 1점부터 5점 사이의 정수를 입력한다. 그다음 서비스 가격을 입력하면, 프로그램은 앞에서 입력한 점수들을 바탕으로 추천 팁 비율을 계산하고, 해당 비율에 따른 추천 팁 금액을 구한다.

계산이 끝나면 프로그램은 추천 팁 비율, 추천 팁 금액, 추천 지불 금액을 차례대로 출력한다. 추천 팁 금액은 실제 지불 상황을 고려하여 100원 단위로 계산되도록 하였다.

또한 잘못된 입력에 대한 예외 처리도 포함하였다. 서비스 종류는 1부터 3 사이의 정수만 입력할 수 있고, 만족도 점수는 1부터 5 사이의 정수만 입력할 수 있다. 가격은 양수로 입력해야 한다. 범위를 벗어난 값이나 정수가 아닌 값이 입력되면 프로그램은 계산을 진행하지 않고 오류 메시지를 출력한다.

## 3. 코드 구조

이 프로그램의 팁 비율 계산은 `TipDetermine`, `Restaurant`, `Delivery` 세 개의 클래스를 중심으로 이루어진다.

`TipDetermine` 클래스는 부모 클래스이며, `Restaurant`와 `Delivery` 클래스가 공통적으로 사용하는 기본 구조를 제공한다. 이 클래스의 자식 클래스로는 식당 홀서비스를 담당하는 `Restaurant` 클래스와 배달 서비스를 담당하는 `Delivery` 클래스가 있다. 식당 홀서비스와 배달 서비스에 해당하지 않는 기타 서비스의 경우에는 부모 클래스인 `TipDetermine`에서 팁 비율을 계산하도록 하였다. 팁 비율은 서비스 만족도, 사용자의 기분 점수, 랜덤 조정값을 바탕으로 계산되며, 구체적인 팁 비율 산정 방식은 클래스마다 다르게 설정하였다.

`Restaurant`와 `Delivery`는 `TipDetermine`를 상속받아 만든 자식 클래스이며, 부모 클래스의 기본 구조를 활용하되 각 서비스 유형에 맞게 `CalculateBase()` 메서드를 다시 정의한다. `Restaurant`는 식당 홀서비스 상황을 반영하기 위해 직원 만족도 점수를 추가로 입력받고, `Delivery`는 배달 서비스에 맞게 팁 비율 산정 방식을 다르게 설정하였다.

프로그램의 실행 부분에서는 먼저 사용자가 서비스 종류를 선택하고, 선택한 서비스 유형에 따라 필요한 점수를 입력받는다. 이후 해당 서비스 유형에 맞는 객체를 생성하고 `CalculateBase()` 메서드를 실행하여 팁 비율을 계산한다. 마지막으로 `getTips()` 메서드를 통해 계산된 팁 비율을 가져온 뒤, 서비스 가격을 기준으로 추천 팁 금액과 최종 지불 금액을 출력한다.

입력 오류를 처리하기 위해 전체 실행 과정은 `try-except` 구조 안에서 진행된다. 정수가 아닌 값을 입력했을 때 발생하는 `ValueError`를 처리하며, 서비스 종류나 점수, 가격이 정해진 범위를 벗어난 경우에는 직접 만든 `InputError` 예외를 발생시켜 처리한다. 이를 통해 잘못된 입력이 들어왔을 때 사용자에게 오류 메시지를 출력하도록 하였다.
